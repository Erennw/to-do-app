# My To-Do List

A simple, modern, and responsive To-Do List web application built with Vanilla JavaScript, HTML5, and CSS3.  
The app saves tasks in LocalStorage, allowing them to persist even after refreshing the page.

---

## Live Demo
View on GitHub Pages: https://erennw.github.io/to-do-app/

---

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by All, Active, or Completed
- Data stored in LocalStorage
- Responsive design for desktop and mobile
- Optional dark mode toggle
- Press Enter to add tasks quickly
- Built with Vanilla JavaScript only (no frameworks)

---

## Technologies Used
- HTML5  
- CSS3 (Flexbox, Responsive Design)  
- JavaScript (ES6+)
